\chapter*{Charte rédactionnelle}
\addcontentsline{toc}{chapter}{Charte rédactionnelle}

\section*{Objectif}

Présenter la balistique de manière scientifique, progressive et accessible.

\section*{Structure des phénomènes}

\begin{enumerate}
\item Intuition
\item Observation pratique
\item Explication physique
\item Modélisation
\item Équations
\item Ordres de grandeur
\item Cas particuliers
\item Références croisées
\item Bibliographie
\end{enumerate}

\section*{Conventions}

Utilisation du système international.

\begin{itemize}
\item Distance : m
\item Vitesse : m/s
\item Masse : kg
\item Temps : s
\item Pression : Pa
\item Énergie : J
\end{itemize}
